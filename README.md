## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
npm start
```

## Deploy on Vercel

The easiest way to deploy this portfolio is to use the [Vercel Platform](https://vercel.com).

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/portfolio-prasad-madusanka)

### Deployment Steps:

1. Push your code to GitHub
2. Import your repository in Vercel
3. Vercel will automatically detect Next.js and configure the build settings
4. Click "Deploy"

## Customization

### Personal Information

Update the following in the components:
- `Hero.tsx` - Your name, title, and introduction
- `About.tsx` - Your bio and personal details
- `Skills.tsx` - Your technical skills
- `Projects.tsx` - Your portfolio projects
- `Contact.tsx` - Your contact information

### Social Links

Update social media links in:
- `Hero.tsx`
- `Footer.tsx`

### Colors and Styling

Modify the color scheme in `globals.css` and component files.
